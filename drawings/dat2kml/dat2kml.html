dat2kml

Convert ISC/AERMOD PLOTFILE result to KML file and regrid to SURFER grd file
ASCII TXT(no matter headerlines), csv (X,Y,C), or a SURFER ASCII .grd file are welcomed.

Example PLOTFILE file might be found here.

Results KML could be read by Leaflet.

PLOTFILE or grd file could be overlayed with OTM at NCLonOTM
